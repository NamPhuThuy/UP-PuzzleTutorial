# UPack-Puzzle-Tutorial

## HOW TO USE

Create a new partial class TutorialAdapter in your project 
Create the scriptableObject TutorialData
The submodule should be placed like this:  

<pre>
_Project/
├── Tutorial_Module/
│   ├── TutorialAdapter.Extend.cs
│   ├── TutorialData.asset
│   └── UPack-Puzzle-Tutorial/
</pre>
